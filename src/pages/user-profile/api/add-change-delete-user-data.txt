ДОБАВЛЕНИЕ адреса:
  передаём вот этот объект:
    const body: Action = {
      action: "addAddress",
      address: {
        streetName: "Example Street",
        streetNumber: "4711",
        postalCode: "80933",
        city: "Exemplary City",
        country: "DE",
      },
    };

УДАЛЕНИЕ адреса:
  передаём вот этот объект:
    const body: Action = {
      action: "removeAddress",
      addressId: "{Rsv8fxK0}",
    };

ИЗМЕНЕНИЕ адреса
  передаём вот этот объект:
    const body: Action = {
      action: "changeAddress",
      addressId: "TcpXmc6m",
      address: {
        streetName: "sovet",
        streetNumber: "999",
        postalCode: "80933",
        city: "Gomel",
        country: "BY",
      },
    };

ПО УМОЛЧАНИЮ адреса
  передаём вот этот объект:
    const body: Action = {
		  "action" : "setDefaultBillingAddress" или "setDefaultShippingAddress",
		  "addressId" : "TcpXmc6m"
	}

Изменение First name:
  передаём вот этот объект:
    const body: Action = {
      "action" : "setFirstName",
      "firstName" : "John"
    }

Изменение Last name
  передаём вот этот объект:
    const body: Action = {
      "action" : "setLastName",
      "lastName" : "Person"
    }

Изменение даты рождения
  передаём вот этот объект:
    const body: Action = {
      "action" : "setDateOfBirth",
      "dateOfBirth" : "2015-10-21"
    }

Изменение еmail
  передаём вот этот объект:
    const body: Action = {
      "action" : "changeEmail",
      "email" : "email@example.com"
    }
